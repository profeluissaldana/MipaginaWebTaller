🌶️ Framework Flask

Introducción al desarrollo Backend con Python.

1. ¿Qué es Flask?

Flask es un Framework Backend desarrollado en Python. Nos permite crear servidores web y conectar nuestras páginas HTML con lógica de programación.

💡 Gracias a Flask podemos:
  • Crear páginas dinámicas.
  • Procesar formularios.
  • Conectar bases de datos.
  • Crear sistemas web reales.

Antes trabajábamos solamente con HTML, CSS y JavaScript en el navegador del usuario (Frontend). Ahora comenzaremos a trabajar también del lado del servidor (Backend).

Flask conecta Python con nuestras páginas web.

2. ¿Qué necesitamos para usar Flask?

Antes de comenzar debemos instalar algunas herramientas fundamentales.

Herramientas necesarias
  • ✅ Python instalado.
  • ✅ Visual Studio Code.
  • ✅ Terminal integrada.
  • ✅ Librería Flask.
⚠️ Flask NO viene instalado por defecto en Python. Debemos instalarlo manualmente usando la terminal.
Python será el motor principal de nuestro servidor.

3. Entorno Virtual de Python (venv)

En programación profesional, cada proyecto debe trabajar con sus propias librerías independientes.

Para lograr esto utilizamos un Entorno Virtual.

💡 Un entorno virtual crea una carpeta aislada donde instalaremos únicamente las dependencias necesarias para ese proyecto.
📦 ¿Por qué es importante usar venv?
  • ✅ Evita conflictos entre proyectos.
  • ✅ Cada proyecto tiene sus propias librerías.
  • ✅ Permite trabajar con distintas versiones.
  • ✅ Facilita subir proyectos a servidores.
  • ✅ Es una práctica profesional real.
🧠 Ejemplo sencillo para entender venv

Imagina que cada proyecto es un laboratorio distinto.

  • 🧪 Un laboratorio puede usar Flask 3.
  • 🧪 Otro puede usar Flask 2.
  • 🧪 Otro puede usar librerías completamente diferentes.

Gracias al entorno virtual, los proyectos no se mezclan entre sí.

🛠️ Crear el entorno virtual

Abrimos la terminal dentro de la carpeta del proyecto y ejecutamos:

python -m venv .venv
🔍 ¿Qué significa?
Python crea una carpeta llamada .venv que contendrá un entorno aislado para este proyecto.
▶️ Activar el entorno virtual

En Windows debemos ejecutar:

.venv\Scripts\activate
⚠️ Cuando el entorno virtual está activo, veremos (.venv) al inicio de la terminal.
⛔ Desactivar el entorno virtual

Cuando terminamos de trabajar podemos salir del entorno usando:

deactivate
🎉 A partir de ahora todas las librerías que instalemos quedarán únicamente dentro de este proyecto.

4. Instalación de Flask

Una vez activado el entorno virtual, ya podemos instalar Flask.

Para instalar Flask abrimos la terminal de Visual Studio Code y escribimos el siguiente comando:

pip install flask
🔍 ¿Qué hace pip?
Pip es el administrador de paquetes de Python. Nos permite descargar librerías externas desde internet.
📦 Verificar si Flask quedó instalado

Podemos comprobarlo usando:

pip show flask
🎉 Si aparece información de Flask en pantalla, entonces la instalación fue correcta.

5. Nuestro primer servidor Flask

El archivo principal de Flask normalmente se llama app.py.

Allí escribiremos el código que controla el servidor.

from flask import Flask, render_template app = Flask(__name__) @app.route('/') def inicio(): return render_template('inicio.html') if __name__ == '__main__': app.run(debug=True)
Flask crea un servidor local para probar nuestros proyectos.

6. Explicación del código principal

Código ¿Qué realiza?
from flask import Flask Importa la librería Flask al proyecto.
app = Flask(__name__) Crea nuestra aplicación web.
@app.route('/') Define la ruta principal del sitio.
render_template() Carga archivos HTML desde la carpeta templates.
app.run(debug=True) Inicia el servidor Flask en modo desarrollo.
