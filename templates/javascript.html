💛 JavaScript

Añadiendo dinamismo, lógica e interactividad al navegador del usuario.

1. ¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite agregar comportamiento e interactividad a las páginas web.

💡 Gracias a JavaScript podemos:
  • Responder a eventos.
  • Modificar contenido HTML.
  • Crear validaciones.
  • Realizar cálculos.
  • Generar efectos dinámicos.

JavaScript funciona directamente dentro del navegador.

📚 Temario Principal
  • Variables y tipos de datos
  • Condicionales (if / else)
  • Bucles (for / while)
  • Manipulación del DOM
  • Funciones en JavaScript
💻 Primer ejercicio

Crear un archivo llamado: 

script.js

Luego escribir: 

let saludo = "Hola alumnos";

console.log(saludo);
✅ Abrir la consola del navegador con F12 para observar el resultado.

3. Condicionales (if / else)

Los condicionales permiten tomar decisiones dentro del programa. 

let edad = 18;

if(edad >= 18){

    console.log("Es mayor de edad");

}else{

    console.log("Es menor de edad");

}
💡 El programa ejecutará diferentes acciones según se cumpla o no una condición.

4. Bucles (for / while)

Los bucles permiten repetir acciones automáticamente.

🔁 Ejemplo con for 
for(let i = 1; i <= 5; i++){

    console.log(i);

}
🔄 Ejemplo con while 
let contador = 1;

while(contador <= 5){

    console.log(contador);

    contador++;

}

6. Funciones en JavaScript

Una función es un bloque de código diseñado para realizar una tarea específica. Se escribe una sola vez y se puede ejecutar (o "invocar") tantas veces como sea necesario.

💡 Las funciones pueden recibir datos para trabajar (llamados parámetros) y pueden devolvernos un resultado final usando la palabra clave return.
🛠️ Estructura básica y parámetros 
// Definimos la función con un parámetro "nombre"
function saludarPersona(nombre) {
    console.log("Hola, " + nombre);
}

// Invocamos la función pasándole diferentes argumentos
saludarPersona("Carlos"); // Imprime: Hola, Carlos
saludarPersona("Sofía");  // Imprime: Hola, Sofía
🔄 Funciones que retornan valores (return) 
// Esta función calcula un dato y lo devuelve
function calcularPrecioFinal(precioOriginal, descuento) {
    let resta = precioOriginal * (descuento / 100);
    return precioOriginal - resta; // Devuelve el resultado
}

// Guardamos lo que devuelve la función en una variable
let totalAPagar = calcularPrecioFinal(1000, 15); 
console.log(totalAPagar); // Imprime: 850
🔍 Explicación línea por línea: Código de la Calculadora Básica

A continuación, analizamos detalladamente el script utilizado al final de nuestro formulario para procesar la suma de dos números:

El código analizado:
<script>
    const botonsumar = document.getElementById("btn_sumar");
    var r = document.getElementById("resultado");

    botonsumar.addEventListener("click", function () {
        const n1 = Number(document.getElementById("num1").value);
        const n2 = Number(document.getElementById("num2").value);
        r.textContent = "Resultado: " + (n1 + n2);
    })
</script>
Análisis paso a paso:
  • 1. Captura de elementos del DOM:
    const botonsumar = document.getElementById("btn_sumar");
    var r = document.getElementById("resultado");
    Buscamos el botón y el encabezado en el HTML mediante sus atributos id únicos para poder controlarlos desde JavaScript.
  • 2. Escuchador de Eventos:
    botonsumar.addEventListener("click", function () { ... })
    Le indicamos al botón que se quede "escuchando". En el momento en que el usuario haga un clic sobre él, se dispara de forma automática la función interna.
  • 3. ¿Por qué usamos .value?
    document.getElementById("num1").value
    Cuando usamos únicamente document.getElementById("num1"), JavaScript obtiene toda la etiqueta HTML completa. Para acceder específicamente al texto o número que el usuario escribió dentro de esa caja de texto, necesitamos llamar a su propiedad interna .value.
  • 4. Conversión con Number():
    Por defecto, todo lo que se recupera de un <input> se recibe como una cadena de texto (String). Al envolverlo en Number(...), transformamos ese texto en un valor numérico real para poder operar matemáticamente.
  • 5. ¿Por qué la suma va entre paréntesis (n1 + n2)?
    r.textContent = "Resultado: " + (n1 + n2);
    En JavaScript, el signo + sirve tanto para sumar números como para concatenar (unir) textos. Al colocar (n1 + n2) entre paréntesis, obligamos al intérprete a resolver primero la operación aritmética interna (la suma) y, recién después, unir el número obtenido al texto de presentación.
📦 Concepto Avanzado: Funciones que devuelven Objetos

Una forma más limpia y profesional de organizar nuestro código es hacer que una función recopile los datos y los devuelva agrupados dentro de un Objeto. Veamos este ejemplo mejorado:

Código optimizado con Objetos:
<script>
    function obtenerNumeros() {
        return {
            n1: Number(document.getElementById("num1").value),
            n2: Number(document.getElementById("num2").value)
        };
    }

    const btnSumar = document.getElementById("btn_sumar");
    const resultado = document.getElementById("resultado");

    btnSumar.addEventListener("click", function () {
        const numeros = obtenerNumeros();
        resultado.textContent = "Resultado: " + (numeros.n1 + numeros.n2);
    });
</script>
⚠️ ¿Qué confunde a los alumnos? Claves para entenderlo:
1. ¿Por qué usamos llaves {} en el return?

En JavaScript, las llaves {} se utilizan para definir un Objeto. Un objeto es como una "supervariable" o una caja compartimentada que nos permite guardar múltiples datos etiquetados en su interior. Al poner return { ... };, la función no está devolviendo un solo número, sino que está empaquetando y enviando la caja completa con todo lo que tiene adentro.

2. ¿Por qué se usa dos puntos : en vez del igual =?

Este es un punto crítico. Cuando creamos variables sueltas en nuestro código usamos el signo igual (ej: let n1 = 10;).

Sin embargo, dentro de un objeto la sintaxis cambia por completo. Los objetos se componen de pares de Clave: Valor de la siguiente forma: 

n1: Number(...)

Aquí, n1 no es una variable independiente; es la etiqueta o propiedad del compartimento dentro del objeto, y los dos puntos : se encargan de asignarle su contenido. Usar un igual = adentro de las llaves del objeto provocaría un error de sintaxis en el navegador.

3. ¿Cómo se lee el valor con el punto (numeros.n1)?

Cuando ejecutamos la línea const numeros = obtenerNumeros();, la variable numeros se convierte en dueña del objeto devuelto. Para poder abrir la caja y sacar los datos de forma individual, usamos la nomenclatura del punto (.). Escribir numeros.n1 significa textualmente: "Entra a la variable 'numeros' y búscame lo que guardaste bajo la etiqueta 'n1'".
