💛 JavaScript

Añadiendo dinamismo, lógica e interactividad al navegador del usuario.

1. ¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite agregar comportamiento e interactividad a las páginas web.

💡 Gracias a JavaScript podemos:
  • Responder a eventos.
  • Modificar contenido HTML.
  • Crear validaciones.
  • Realizar cálculos.
  • Generar efectos dinámicos.

JavaScript funciona directamente dentro del navegador.

📚 Temario Principal
  • Variables y tipos de datos
  • Condicionales (if / else)
  • Bucles (for / while)
  • Manipulación del DOM
  • Funciones en JavaScript
💻 Primer ejercicio

Crear un archivo llamado: 

script.js

Luego escribir: 

let saludo = "Hola alumnos";

console.log(saludo);
✅ Abrir la consola del navegador con F12 para observar el resultado.

2. Variables y tipos de datos

Las variables permiten almacenar información.

⚠️ Una variable funciona como una caja donde guardamos datos.

Number (número) 25
Boolean (verdadero/falso) true

3. Condicionales (if / else)

Los condicionales permiten tomar decisiones dentro del programa. 

let edad = 18;

if(edad >= 18){

    console.log("Es mayor de edad");

}else{

    console.log("Es menor de edad");

}
💡 El programa ejecutará diferentes acciones según se cumpla o no una condición.
💻 Ejercicio práctico

Crear un programa que:

  • Guarde una nota numérica.
  • Si la nota es mayor o igual a 6: mostrar "Aprobado".
  • Si no: mostrar "Desaprobado".

4. Bucles (for / while)

Los bucles permiten repetir acciones automáticamente.

🔁 Ejemplo con for 
for(let i = 1; i <= 5; i++){

    console.log(i);

}
🔄 Ejemplo con while 
let contador = 1;

while(contador <= 5){

    console.log(contador);

    contador++;

}

6. Funciones en JavaScript

Una función es un bloque de código diseñado para realizar una tarea específica. Se escribe una sola vez y se puede ejecutar (o "invocar") tantas veces como sea necesario.

💡 Las funciones pueden recibir datos para trabajar (llamados parámetros) y pueden devolvernos un resultado final usando la palabra clave return.
🛠️ Estructura básica y parámetros 
// Definimos la función con un parámetro "nombre"
function saludarPersona(nombre) {
    console.log("Hola, " + nombre);
}

// Invocamos la función pasándole diferentes argumentos
saludarPersona("Carlos"); // Imprime: Hola, Carlos
saludarPersona("Sofía");  // Imprime: Hola, Sofía
🔄 Funciones que retornan valores (return) 
// Esta función calcula un dato y lo devuelve
function calcularPrecioFinal(precioOriginal, descuento) {
    let resta = precioOriginal * (descuento / 100);
    return precioOriginal - resta; // Devuelve el resultado
}

// Guardamos lo que devuelve la función en una variable
let totalAPagar = calcularPrecioFinal(1000, 15); 
console.log(totalAPagar); // Imprime: 850
🧮 Proyecto: Funciones con Parámetros y Objetos mediante `if / else`

En esta aplicación integramos un casillero dedicado para el resultado (utilizando un input con el atributo readonly para evitar que se pueda editar manualmente) y el uso de parámetros para controlar qué operación aritmética realizar a partir del mismo objeto contenedor.

Código Base del Archivo (Enfoque tradicional):
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Calculadora Básica</title>
    <style>
        input {
            width: 80px;
            background-color: black;
            color: white;
            font-weight: bold;
            text-align: center;
        }
        button {
            cursor: pointer;
            margin-right: 5px;
        }
    </style>
</head>
<body>
    <h2>Calculadora de Operaciones</h2>

    <form id="formulario">
        <label>Número 1:</label>
        <input type="number" id="num1" autofocus>
        <br><br>

        <label>Número 2:</label>
        <input type="number" id="num2">
        <br><br>

        <button type="button" id="btn_sumar">Sumar</button>
        <button type="button" id="btn_restar">Restar</button>
        <button type="button" id="btn_limpiar">Limpiar</button>
        <br><br>

        <label>Resultado:</label>
        <input type="text" id="resultado" readonly>
    </form>

    <script>
        function obtenerNumeros() {
            return {
                n1: Number(document.getElementById("num1").value),
                n2: Number(document.getElementById("num2").value)
            };
        }

        // Función dinámica tradicional utilizando condicionales 'if'
        function operar(tipoOperacion) {
            const numeros = obtenerNumeros();
            const inputResultado = document.getElementById("resultado");

            if (tipoOperacion === "sumar") {
                inputResultado.value = numeros.n1 + numeros.n2;
            } else if (tipoOperacion === "restar") {
                inputResultado.value = numeros.n1 - numeros.n2;
            }
        }

        // Escuchadores de eventos
        document.getElementById("btn_sumar").addEventListener("click", function() {
            operar("sumar");
        });

        document.getElementById("btn_restar").addEventListener("click", function() {
            operar("restar");
        });

        // Evento de Limpieza
        document.getElementById("btn_limpiar").addEventListener("click", function() {
            document.getElementById("num1").value = "";
            document.getElementById("num2").value = "";
            document.getElementById("resultado").value = "";
            document.getElementById("num1").focus();
        });
    </script>
</body>

🔀 Una Alternativa más Limpia y Clara: La Estructura `switch / case`

A medida que una aplicación incorpora más opciones de cálculo, el uso excesivo de bloques if / else if sucesivos puede ensuciar la legibilidad de nuestro algoritmo. Para solucionar este problema informático, JavaScript ofrece la estructura switch / case.

💡 Ventaja clave: Evalúa una variable directamente contra múltiples opciones fijas, haciendo que el código sea mucho más scannable y fácil de expandir en el futuro.

Si reemplazáramos la lógica de la función operar anterior por un switch, el script se vería de la siguiente forma (restringido únicamente para sumar y restar): 

function operar(tipoOperacion) {
    const numeros = obtenerNumeros();
    const inputResultado = document.getElementById("resultado");

    // Evaluamos el parámetro directamente de forma más ordenada
    switch (tipoOperacion) {
        case "sumar":
            inputResultado.value = numeros.n1 + numeros.n2;
            break;
        case "restar":
            inputResultado.value = numeros.n1 - numeros.n2;
            break;
        default:
            inputResultado.value = "";
    }
}

Componentes obligatorios del switch:

  • case "valor": Define la condición exacta a verificar.
  • break; Es un freno crítico. Le dice a JavaScript que frene la ejecución y salga de la estructura una vez hallada la coincidencia. Si se omite, el código continuará ejecutando los casos de abajo en cascada.
  • default: Funciona como un "else" final de seguridad en caso de que ningún valor coincida.
💪 Desafío Práctico: Expandiendo la Calculadora (Elegir Estructura)

🎯 Consignas para la actividad en clase:

  2. Gran Reto Algorítmico: Añade la operación de Dividir. Debes implementar una validación condicional que inspeccione si el segundo número ingresado es un cero. De ser así, se le debe prohibir la operación al usuario enviándole un mensaje claro mediante un cuadro de texto emergente (usando alert()) y definiendo el casillero del resultado como "Error".
