💛 JavaScript

Añadiendo dinamismo, lógica e interactividad al navegador del usuario.

1. ¿Qué es JavaScript?

JavaScript es un lenguaje de programación que permite agregar comportamiento e interactividad a las páginas web.

💡 Gracias a JavaScript podemos:
  • Responder a eventos.
  • Modificar contenido HTML.
  • Crear validaciones.
  • Realizar cálculos.
  • Generar efectos dinámicos.

JavaScript funciona directamente dentro del navegador.

📚 Temario Principal
  • Variables y tipos de datos
  • Condicionales (if / else)
  • Bucles (for / while)
  • Manipulación del DOM
  • Funciones en JavaScript
💻 Primer ejercicio

Crear un archivo llamado: 

script.js

Luego escribir: 

let saludo = "Hola alumnos";

console.log(saludo);
✅ Abrir la consola del navegador con F12 para observar el resultado.

2. Variables y tipos de datos

Las variables permiten almacenar información.

⚠️ Una variable funciona como una caja dongrade guardamos datos. 
let nombre = "Luis";

let edad = 35;

let profesor = true;
Tipo de dato Ejemplo
String (texto) "Hola"
Number (número) 25
Boolean (verdadero/falso) true
✍️ Actividad práctica

Crear variables para:

  • Nombre del alumno.
  • Edad.
  • Curso.
  • Si le gusta programación (true/false).
🧠 Pongámonos a prueba
¿Qué tipo de dato es "Hola Mundo"?
  • ❌ Number

3. Condicionales (if / else)

Los condicionales permiten tomar decisiones dentro del programa. 

let edad = 18;

if(edad >= 18){

    console.log("Es mayor de edad");

}else{

    console.log("Es menor de edad");

}
💡 El programa ejecutará diferentes acciones según se cumpla o no una condición.
💻 Ejercicio práctico

Crear un programa que:

  • Guarde una nota numérica.
  • Si la nota es mayor o igual a 6: mostrar "Aprobado".
  • Si no: mostrar "Desaprobado".

¿Qué palabra se utiliza para una condición?
  • ✅ if
  • ❌ for
  • ❌ let

4. Bucles (for / while)

Los bucles permiten repetir acciones automáticamente.

🔁 Ejemplo con for 
for(let i = 1; i <= 5; i++){

    console.log(i);

}
🔄 Ejemplo con while 
let contador = 1;

while(contador <= 5){

    console.log(contador);

    contador++;

}
✍️ Actividad práctica

Crear un bucle que muestre:

  • Los números del 1 al 10.
  • Los números pares del 2 al 20.
🧠 Pregunta rápida
¿Para qué sirven los bucles?
  • ❌ Crear imágenes
  • ✅ Repetir acciones
  • ❌ Diseñar páginas
  • ❌ Conectar internet

5. Manipulación del DOM

El DOM representa la estructura HTML de la página.

💡 Gracias al DOM, JavaScript puede modificar elementos HTML en tiempo real. 
document.getElementById("titulo")
.innerText = "Hola Mundo";

Con esto podemos:

  • Cambiar textos.
  • Modificar colores.
  • Mostrar u ocultar elementos.
  • Responder a botones.
💻 Práctica DOM

Crear:

  • Un título con id="titulo".
  • Un botón.
  • Al hacer click: cambiar el texto del título.

6. Funciones en JavaScript

Una función es un bloque de código diseñado para realizar una tarea específica. Se escribe una sola vez y se puede ejecutar (o "invocar") tantas veces como sea necesario.

💡 Las funciones pueden recibir datos para trabajar (llamados parámetros) y pueden devolvernos un resultado final usando la palabra clave return.
🛠️ Estructura básica y parámetros 
// Definimos la función con un parámetro "nombre"
function saludarPersona(nombre) {
    console.log("Hola, " + nombre);
}

// Invocamos la función pasándole diferentes argumentos
saludarPersona("Carlos"); // Imprime: Hola, Carlos
saludarPersona("Sofía");  // Imprime: Hola, Sofía
🔄 Funciones que retornan valores (return) 
// Esta función calcula un dato y lo devuelve
function calcularPrecioFinal(precioOriginal, descuento) {
    let resta = precioOriginal * (descuento / 100);
    return precioOriginal - resta; // Devuelve el resultado
}

// Guardamos lo que devuelve la función en una variable
let totalAPagar = calcularPrecioFinal(1000, 15); 
console.log(totalAPagar); // Imprime: 850
💻 Desafío de Funciones

Consigna: El Validador de Contraseñas

Crea una función llamada validarClave(password) que reciba un texto. Si el texto tiene 8 o más caracteres (puedes usar password.length), la función debe retornar true. Si tiene menos, debe retornar false.

🚀 ¡Reto Informático!: Conecta esta función a un <input type="password"> de HTML y haz que el borde del input cambie a color verde si es segura, o a rojo si es insegura al presionar un botón.
