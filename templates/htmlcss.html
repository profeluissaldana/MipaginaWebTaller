🌐 HTML5 y CSS3

Los cimientos de la Web: estructura, diseño y páginas modernas.

¡Aviso importante! HTML estructura el contenido y CSS le da diseño visual. Ambos trabajan juntos en todos los sitios web modernos.

1. ¿Qué es HTML?

HTML significa: HyperText Markup Language.

Es el lenguaje encargado de estructurar el contenido de una página web.

💡 HTML NO es un lenguaje de programación.
  • Organiza contenido.
  • Define títulos.
  • Agrega imágenes.
  • Crea enlaces.
  • Construye formularios.

Todo lo que vemos en una página web comienza con HTML.

HTML construye la estructura principal del sitio web.
🧪 Actividad práctica

Crear un archivo llamado index.html y escribir: 

<h1>Mi primera página web</h1>

<p>Hola profesor</p>
✅ Guardar el archivo y abrirlo desde el navegador.
🧠 Comprobamos lo aprendido
¿Qué función cumple HTML?
  • ❌ Crear bases de datos
  • ❌ Programar videojuegos
  • ✅ Estructurar páginas web
  • ❌ Crear sistemas operativos

2. Estructura básica de un documento HTML

Todo documento HTML posee una estructura principal. 

<!DOCTYPE html>
<html lang="es">

<head>

    <meta charset="UTF-8">

    <meta name="viewport"
          content="width=device-width, initial-scale=1.0">

    <title>Mi primera página</title>

</head>

<body>

    <h1>Hola Mundo</h1>

    <p>Mi primera página web.</p>

</body>

</html>
🎉 Esta es la base mínima que utiliza cualquier página web.
💻 Práctica en Visual Studio Code

Crear un documento HTML completo utilizando:

  • <head>
  • <body>
  • <h1>
  • <p>
⚠️ Revisar correctamente la indentación.
🧠 Pregunta rápida
¿Dónde colocamos el contenido visible?
  • ❌ Dentro de <head>
  • ✅ Dentro de <body>
  • ❌ Dentro de <meta>
  • ❌ Dentro de <title>

3. La etiqueta <head>

El <head> contiene información interna de la página.

Esa información normalmente NO aparece visible en pantalla.

Etiqueta ¿Para qué sirve?
<title> Define el título de la pestaña del navegador.
meta charset="UTF-8" Permite usar letras especiales como ñ, á, é.
viewport Hace que la página sea responsive.
description Describe la página para buscadores.
favicon Agrega el ícono pequeño de la pestaña. 
<html lang="es">
✍️ Ejercicio

Cambiar el idioma del documento de: 

lang="en"

a: 

lang="es"
🧠 Verificación rápida
¿Qué etiqueta define el título de la pestaña?
  • ❌ <meta>
  • ✅ <title>
  • ❌ <body>
  • ❌ <strong>

4. Etiquetas fundamentales

Etiqueta Función
<h1> hasta <h6> Títulos y subtítulos.
<p> Párrafos.
<strong> Texto importante.
<em> Texto en cursiva.
<span> Modificar una parte específica. 
<p>

Hola
<span style="color:red">Mundo</span>

</p>
💻 Actividad práctica

Crear:

  • Un título principal.
  • Dos subtítulos.
  • Un párrafo.
  • Una palabra resaltada con <strong>.
🧠 Pongámonos a prueba
¿Qué etiqueta se utiliza para resaltar texto importante?
  • ❌ <hr>
  • ❌ <span>
  • ✅ <strong>
  • ❌ <br>

5. Formas de agregar CSS

❌ Inline 
<p style="color:red">
Texto
</p>
⚠️ Embebido 
<style>

p{
    color:blue;
}

</style>
✅ Archivo externo 
<link rel="stylesheet"
href="css/estilos.css">
🎨 Práctica CSS

Crear un archivo estilos.css y cambiar:

  • Color del fondo.
  • Color del texto.
  • Tamaño del título.
🧠 Repaso final
¿Cuál es la forma MÁS recomendada de usar CSS?
  • ❌ Inline
  • ❌ Embebido
  • ✅ Archivo externo
  • ❌ Ninguna
